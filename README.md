# Project 1 - Rule Based AI Chatbot

## Description
This project is a simple Rule Based AI Chatbot built using Python.  
It responds to user inputs based on predefined rules and conditions.

## Features
- User input handling
- Rule-based response system
- Simple conversational chatbot behavior

## How it works
The chatbot checks user input and matches it with predefined rules.  
Based on the match, it returns the appropriate response.

## Tools Used
- Python
- Pydroid 3 (Mobile IDE)
## Output
The chatbot interacts with users and gives relevant responses based on rules.

## Note
This is a basic AI chatbot project for learning purposes.
